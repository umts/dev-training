    class UMTSTraining::MessageCatalog

    Essentially a hash with two additions:

    1. It includes the IndifferentAccess, MergeInitializer, and MethodAccess mixins from Hashie.

    2. It can also be initialized with a File-ish object that contains a YAML mapping.
