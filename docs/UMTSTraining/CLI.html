    class UMTSTraining::CLI

    The UMTSTraining::CLI class is for interacting with the user via the command-line. It is a subclass of Highline with a few minor additions.

    Public Class Methods

    new(messages_file, input: $stdin, output: $stdout) click to toggle source

    Create a new cli instance. message_file is typically a File-ish object that contains the YAML messages that can be displayed to the user. It can, however, be anything that MessageCatalog.new will accept.

    Calls superclass method

def initialize(messages_file, input: $stdin, output: $stdout)
  super(input, output, 80)
  @message_catalog = MessageCatalog.new(messages_file)
end

    message(name) click to toggle source

    say the message out of the message catalog with the name, name.

    Note also that this class delegates methods to said MessageCatalog, so CLI#my_message is equivalent to CLI#message(:my_message)

    # File lib/umts_training/cli.rb, line 28
def message(name)
  say @message_catalog.fetch(name)
end
    pw_ask(prompt) click to toggle source

    Ask for a password. This is a wrapper around HighLine#ask that displays *s as you type.

    # File lib/umts_training/cli.rb, line 43
def pw_ask(prompt)
  ask(prompt) { |q| q.echo = '*' }
end
